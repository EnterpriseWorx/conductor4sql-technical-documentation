Central Console Overview

The Central Console is used for monitoring and controlling all clients installed in your Conductor4SQL environment.

To Open the Central Console click on the Central tab on the ribbon

The Central Console is divided up into a number of ribbon buttons:

  • Central Client Dashboard - Check if there is an issue in your environment with a glance

  • Central Client Overview - A detailed overview of every installed client

  • Central Client Control - multiple user query, user command, meta data compare and file send

  • Manage Data Synchronisation - pull and push data between clients and central

  • Manage Scripts - Manage and maintain saved user commands and queries

  • Auditing - Monitor DDL changes for all installed clients
